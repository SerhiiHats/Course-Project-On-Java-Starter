Тема курсового проекта: 
"Создайте программу, которая выводит в консоль число имени, например, на вход Roman = R(18) +
О(15) + M(13) + A(1) + N(14) = 61 на выход 61."


Мною написан проект NameNumber в котором создан класс NameNumber в нем написан метод nameToNumber.
В методе main класса NameNumber пользователю предлагается ввести имя что б получить число имени.
После ввода имени - это имя перебирается в цикле по символам, каждый символ передается в метод nameToNumber в 
котором инициализирован одномерный массив с английским алфавитом, все символы с алфавита перебираются в цикле и сравниваются с символом полученім из имени,
при тождестве возвращает в метод main индекс символа в массиве алфавита.
В методе main int индекса всех символом в имени сумируются.
Первоначальное имя введенное пользователем и полученная сумма индексов символов имени, являющихся числом имени, выводятся в консоль.
Также организован цикл с возможность неоднократного ввода имени и получением числа имени, 
выход из цикла организован по вводу в консоль цифры "0". Информационное сообщение об этом выводится в консоль.
