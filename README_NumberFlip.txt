Цель: Продемонстрировать навыки с базовыми составляющими Java.


Тема курсового проекта: 
"Напишите метод, который принимает число, которое не содержит нулей и возвращает цифры в
обратном порядке.
 Например, на вход: 789 на выход: 987"


Мною написан проект NumberFlip в котором создан клас NumberFlip в нем написан метод flip
который принимает целое число введенное пользователем с консоли (кроме этого организована проверка на правильность
ввода пользователем числа) и возвращает это же число цифрами в обратном порядке (реверс числа).
Первоначальное число введенное пользователем и полученное инвертированое число выводятся в консоль.
Также организован цикл с возможность неоднократного ввода числа пользователем и получением инвертируемого (реверса) этого числа, 
выход из цикла организован по вводу в консоль цифры "0". Информационное сообщение об этом выводится в консоль.
