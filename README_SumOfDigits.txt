Цель: Продемонстрировать навыки с базовыми составляющими Java.


Тема курсового проекта: 
"Напишите метод, который принимает число и возвращает сумму цифр данного числа"


Мною написан проект SumOfDigits в котором создан клас SumOfDigits в нем написан метод digits
который принимает целое число введенное пользователем с консоли (кроме этого организована проверка на правильность
ввода пользователем числа) и возвращает сумму цифр данного числа.
Первоначальное число введенное пользователем и полученная сумма цифр данного числа выводятся в консоль.
Также организован цикл с возможность неоднократного ввода числа пользователем и получением суммы его цифр, 
выход из цикла организован по вводу в консоль цифры "0". Информационное сообщение об этом выводится в консоль.
