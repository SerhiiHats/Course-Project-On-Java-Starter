Цель: Продемонстрировать навыки с базовыми составляющими Java.


Тема курсового проекта: 
"Создайте программу, которая выводит в консоль число даты. Например, на вход 02/11/1995 = 0 +
2 + 1 + 1 + 1 + 9 + 9 + 5 = 28 на выход = 28"


Мною написан проект NumberOfDates в котором создан класс NumberOfDates в нем написан метод dateconvert.
В методе main класса NumberOfDates переменной data типа String присвоено предустановленное тестовое значение даты "02/11/1995" 
это значение даты выводится в консоль, после этого указаное значение даты передается в метод dateconvert
в котором подсчитываются суммы всех цифр этой даты и уже сумма всех цифр этой даты возвращается назад в метод main и выводится в консоль.

Далее для усложнения и отображения реальности была применена реальная текущая дата времени полученая в классе Date, после чего она была сформатированна
конструктором класса SimpleDateFormat до необходимого нам формата как указано в задании: "dd/MM/yyyy" и переведена в значение типа String
для дальнейшей её обработки. После чего с текущей датой были проведены те же действия что и абзацем выше, для реальности и наглядности работоспособности кода, 
но уже с реальной датой.
